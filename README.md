# RxSwiftPractice
学习RxSwift后的练习项目，会写多少个暂时未定。

使用的版本为：
```
pod 'RxSwift', '~> 4.0.0'
pod 'RxCocoa', '~> 4.0.0'
```

## 项目1：TapCounter
计算点击次数。

学习如何订阅按钮点击事件，以及数据和界面绑定，当数据改变，界面跟着改变。
